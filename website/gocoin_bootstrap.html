Bootstraping

Since Gocoin's client is a so-called full node, the initial block chain synchronization is quite a time consuming operation.

With a good internet connection you may speed up the process quite much by starting the client with -v -trust -nowallet switches.

Switch -v causes the node to only maintain UTXO database in memory. It writes the content of the database to disk when you are shutting down the client properly.

Switch -trust makes the node to treat each new downloaded block as trusted, thus not verifying the scripts inside its transactions. After syncing your chain with this switch on, it is important to check the hash of the last received block (e.g. comparing it with a value from a trusted block explorer).

Switch -nowallet disables wallet functionality which speeds up block processing. It also limit the memory used the node.
