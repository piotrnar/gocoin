Installation

System requirements

client

If you want to run the full node (client), your system ought to have at least 16GB of RAM.
It is possible to run it will less memory (e.g. 8GB), but it will likely affect performance and might even make the software to crash sometimes.

To run the client you also need 64-bit OS and a file system that support files larger than 4GB (e.g NTFS or ext2/ext3/ext4).

wallet

If you only want to use the wallet with the corresponding balio tool, you don't have to worry about the system memory or a file system.

Download binaries

The binaries (not always the latest version) for Windows and Linux OS can be downloaded from sourceforge.net

Build from source

The process is described in the main README.md file of the github repo piotrnar/gocoin
